# corebuttons.css

A lightweight, modern CSS button utility framework by [@a_telegram_user](https://t.me/a_telegram_user)

## Usage


```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/homo-developer/web-frame/corebuttons.css">
```

## Button Classes

- `btn btn-primary`
- `btn btn-secondary`
- `btn btn-outline`
- `btn btn-danger`
